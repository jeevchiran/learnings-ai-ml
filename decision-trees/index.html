Decision Trees and Ensemble Methods

A rigorous, interactive path from single decision trees through gradient boosting. Every concept is grounded in a real-world problem before formulas appear. Each module follows a Why → What → How arc with worked examples, interactive widgets, and practice problems at multiple difficulty levels.

Learning Path

Study the modules in order. The first four modules build the decision tree from scratch — intuition, math, and failure modes. The final three show how ensembles fix those failure modes at scale.

  • Module 1: Decision Trees — Intuition and Structure
    Why trees? How a decision tree partitions the feature space, makes predictions, and represents a piecewise-constant function. Anatomy of nodes, branches, leaves, and depth.
  • Module 2: Impurity Measures and the Training Process
    Gini impurity, Shannon entropy, and misclassification rate — derived from first principles. Information gain and Gini gain. The CART algorithm: recursive binary splitting, stopping criteria, and computational complexity.
  • Module 3: Gini Index Splits — Finding the Optimal Threshold
    Full worked example of finding the best split on a 12-sample dataset. Handling continuous and categorical features. Comparing Gini and entropy splits in practice.
  • Module 4: Advantages, Limitations, and Pruning
    Strengths (interpretability, no scaling, mixed types, interaction capture). Failure modes (variance, instability, greedy splits). Pre-pruning and cost-complexity post-pruning with the math.
  • Module 5: Bagging and Random Forests
    The variance problem with single trees. Bootstrap aggregating. Random forests: decorrelating trees through random feature subsets. Feature importance, out-of-bag error, and the variance reduction derivation.
  • Module 6: Gradient Boosting
    Boosting for bias reduction. Gradient boosting as functional gradient descent. Pseudo-residuals, loss functions (MSE and log-loss), shrinkage. XGBoost second-order approximation and regularization.
  • Module 7: Hyperparameter Tuning
    Every knob for Random Forests and Gradient Boosting explained through the bias-variance lens. Practical tuning strategy, interaction effects (learning rate vs n_estimators), and interactive simulators.

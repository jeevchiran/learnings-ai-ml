Natural Language Processing

A 10-module interactive path through Natural Language Processing — from raw text to language models. Starting with how computers read and clean text, building up to syntactic and semantic analysis, then text representations and generative models. Each module follows a Why → What → How arc with interactive widgets, Python code (spaCy and NLTK), and practice problems.

Learning Path

10 modules · spaCy & NLTK · Interactive widgets

  • Module 1: Text Normalization & Edit Distance Lexical
    Tokenization, lowercasing, stopword removal, and the Levenshtein edit distance algorithm. Interactive matrix widget shows the DP computation step by step.
  • Module 2: Stemming, Lemmatization & Noisy Channel Lexical
    Morphological analysis: Porter and Snowball stemmers vs lemmatization. The Bayesian noisy channel model for spell correction. Step-through widget for P(w|e) = P(e|w)P(w).
  • Module 3: POS Tagging & Shallow Parsing Syntactic
    Penn Treebank tag set, rule-based and HMM-based taggers, NP chunking with chunk grammar. Live POS tagger colors tokens by grammatical category.
  • Module 4: Context-Free Grammars, Parse Trees & Deep Parsing Syntactic
    Formal CFG definition G=(N,Σ,R,S), constituency trees, structural ambiguity, and grammatical agreement. SVG step-through parse tree builder.
  • Module 5: Lexical Semantics & Word Sense Disambiguation Semantic
    Polysemy vs homonymy, WordNet synsets, hypernyms and hyponyms, the Lesk algorithm. Interactive WordNet hierarchy explorer and Lesk step-through.
  • Module 6: Distributional Semantics & PMI Semantic
    Distributional hypothesis, co-occurrence matrices, PMI formula, PPMI, cosine similarity. Live PMI heatmap (Plotly) from user-editable corpus.
  • Module 7: NER, IOB Labeling, CRF, SRL & Coreference Semantic
    Named entity types, BIO tagging scheme, CRF sequence models, semantic role labeling, and coreference resolution. Live NER highlighter and IOB step-through.
  • Module 8: Text Representation — BoW, TF-IDF & Topic Modeling Representation
    Bag of words, vocabulary construction, TF-IDF formula, IDF weighting, and NMF topic modeling. Live TF-IDF calculator with Plotly heatmap.
  • Module 9: Word Embeddings Representation
    word2vec (skip-gram and CBOW), the embedding space, cosine similarity, analogy tasks, and contextual embeddings (BERT overview). Word similarity explorer widget.
  • Module 10: N-gram Language Models & Applications Use Cases
    Chain rule, Markov assumption, MLE estimation, Laplace smoothing, perplexity, text generation, extractive summarization, and MT overview. N-gram generator and perplexity calculator widgets.

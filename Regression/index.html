Regression: From Theory to Practice

A comprehensive, interactive learning path covering linear and logistic regression -- from intuition through full mathematical derivation to real-world application.

Learning Path

These modules are designed to be studied in sequence. Each builds on the previous, following a deliberate arc: Why do we need this? What is it? How does it work?

  • Module 1: Linear Regression
    The foundational model for continuous outcomes. Assumptions, coefficient interpretation, performance evaluation, and the house price prediction problem.
  • Module 2: Mathematics of Linear Regression
    Full derivation from first principles. The normal equation, gradient descent with animated loss surfaces, and the Gauss-Markov theorem.
  • Module 3: Logistic Regression
    Extending regression to classification. Binary and multiclass, decision boundaries, regularization (L1/L2), hyperparameter tuning. Spam, diabetes, churn.
  • Module 4: Mathematics of Logistic Regression
    MLE derivation, cross-entropy loss, gradient computation, Newton's method vs gradient descent. Bayesian interpretation of regularization.

Hands-On Labs (Jupyter)

  • Lab 1: Linear Regression from Scratch
    Implement OLS and gradient descent in NumPy. Apply to Ames Housing dataset. Residual diagnostics.
  • Lab 2: Logistic Regression from Scratch
    Implement MLE, apply to customer churn and MNIST digits. Regularization paths and cross-validation.

Module 2: Mathematics of Linear Regression

Bridge from Module 1: In Module 1, we fit a line by minimizing the sum of squared residuals. We stated the result; now we derive it from first principles. Why does OLS work? How do we compute the optimal parameters analytically and iteratively? What guarantees does the solution carry?

The Cost Function

We seek parameters \boldsymbol{\beta} \in \mathbb{R}^{p+1} that minimize the mean squared error. In scalar form:

J(\boldsymbol{\beta}) = \frac{1}{2n} \sum_{i=1}^{n} \left( y_i - \mathbf{x}_i^T \boldsymbol{\beta} \right)^2

In matrix notation, let \mathbf{X} \in \mathbb{R}^{n \times (p+1)} be the design matrix (with a column of ones for the intercept), and \mathbf{y} \in \mathbb{R}^n the response vector:

J(\boldsymbol{\beta}) = \frac{1}{2n} (\mathbf{y} - \mathbf{X}\boldsymbol{\beta})^T (\mathbf{y} - \mathbf{X}\boldsymbol{\beta})

Why Squared Errors?

If we assume the errors are i.i.d. Gaussian, \varepsilon_i \sim \mathcal{N}(0, \sigma^2), then maximizing the likelihood of the observed data is equivalent to minimizing the sum of squared residuals. The MLE and OLS solutions coincide under Gaussian noise.

\log L(\boldsymbol{\beta}) = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^n(y_i - \mathbf{x}_i^T\boldsymbol{\beta})^2

Maximizing this with respect to \boldsymbol{\beta} is equivalent to minimizing \sum(y_i - \mathbf{x}_i^T\boldsymbol{\beta})^2.

The Normal Equation (Analytical Solution)

We derive the closed-form solution by setting the gradient of J to zero.

1
J(\boldsymbol{\beta}) = \frac{1}{2n}(\mathbf{y} - \mathbf{X}\boldsymbol{\beta})^T(\mathbf{y} - \mathbf{X}\boldsymbol{\beta})
Expand the quadratic form.
2
= \frac{1}{2n}\left(\mathbf{y}^T\mathbf{y} - 2\boldsymbol{\beta}^T\mathbf{X}^T\mathbf{y} + \boldsymbol{\beta}^T\mathbf{X}^T\mathbf{X}\boldsymbol{\beta}\right)
Using (\mathbf{X}\boldsymbol{\beta})^T = \boldsymbol{\beta}^T\mathbf{X}^T and the symmetry of the scalar \boldsymbol{\beta}^T\mathbf{X}^T\mathbf{y} = \mathbf{y}^T\mathbf{X}\boldsymbol{\beta}.
3
\nabla_{\boldsymbol{\beta}} J = \frac{1}{2n}\left(-2\mathbf{X}^T\mathbf{y} + 2\mathbf{X}^T\mathbf{X}\boldsymbol{\beta}\right) = \frac{1}{n}\left(\mathbf{X}^T\mathbf{X}\boldsymbol{\beta} - \mathbf{X}^T\mathbf{y}\right)
Differentiate using matrix calculus rules: \frac{\partial}{\partial \mathbf{a}}(\mathbf{a}^T\mathbf{b}) = \mathbf{b} and \frac{\partial}{\partial \mathbf{a}}(\mathbf{a}^T\mathbf{A}\mathbf{a}) = 2\mathbf{A}\mathbf{a} for symmetric \mathbf{A}.
4
\nabla_{\boldsymbol{\beta}} J = 0 \implies \mathbf{X}^T\mathbf{X}\boldsymbol{\beta} = \mathbf{X}^T\mathbf{y}
Set gradient to zero. These are the normal equations.
5
\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}
Multiply both sides by (\mathbf{X}^T\mathbf{X})^{-1}, which exists when \mathbf{X} has full column rank (no perfect multicollinearity).

Geometric Interpretation: Projection

The fitted values \hat{\mathbf{y}} = \mathbf{X}\hat{\boldsymbol{\beta}} = \mathbf{X}(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y} are the orthogonal projection of \mathbf{y} onto the column space of \mathbf{X}. The matrix \mathbf{H} = \mathbf{X}(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T is the "hat matrix" -- it puts the hat on \mathbf{y}.

The residual vector \mathbf{e} = \mathbf{y} - \hat{\mathbf{y}} is perpendicular to the column space: \mathbf{X}^T\mathbf{e} = 0. This is exactly what the normal equations state.

Gradient Descent

When \mathbf{X}^T\mathbf{X} is very large or ill-conditioned, we use an iterative approach instead of computing the inverse directly.

The Algorithm

1

Initialize \boldsymbol{\beta}^{(0)} (often zeros).

2

Compute gradient: \nabla J = \frac{1}{n}\mathbf{X}^T(\mathbf{X}\boldsymbol{\beta}^{(t)} - \mathbf{y})

3

Update: \boldsymbol{\beta}^{(t+1)} = \boldsymbol{\beta}^{(t)} - \alpha \nabla J

where \alpha > 0 is the learning rate (step size).
4

Repeat until \|\nabla J\| < \epsilon (convergence).

Why It Works: Convexity

J(\boldsymbol{\beta}) is a convex function (the Hessian \frac{1}{n}\mathbf{X}^T\mathbf{X} is positive semi-definite). Any local minimum is the global minimum. Gradient descent with a sufficiently small step size is guaranteed to converge.

Learning Rate Effects

Learning RateBehavior
Too small (\alpha \ll 1)Converges but extremely slowly -- thousands of iterations needed
Just rightSmooth convergence in tens to hundreds of iterations
Too large (\alpha > \frac{2}{\lambda_{\max}})Diverges -- parameters oscillate or explode to infinity

Application: Advertising Budget Optimization

Problem

A company spends on TV and online advertising. We model revenue as:

\text{Revenue} = \beta_0 + \beta_1 \cdot \text{TV} + \beta_2 \cdot \text{Online}

Given 20 observations, compute \hat{\boldsymbol{\beta}} via the normal equation.

Working Through the Normal Equation

1

Form the design matrix \mathbf{X} \in \mathbb{R}^{20 \times 3} (ones column, TV, Online) and response \mathbf{y} \in \mathbb{R}^{20}.

2

Compute \mathbf{X}^T\mathbf{X} (a 3 \times 3 symmetric matrix) and \mathbf{X}^T\mathbf{y} (a 3 \times 1 vector).

3

Solve: \hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y} = \begin{pmatrix} 3.2 \\ 0.045 \\ 0.18 \end{pmatrix}

Interpretation: baseline revenue is $3.2M, each $1K on TV adds $45 in revenue, each $1K online adds $180.
4

Optimization: with $100K budget, allocate to maximize 0.045 \cdot \text{TV} + 0.18 \cdot \text{Online}. Since online ROI is 4x that of TV, allocate the full budget to online (in this simplified model). In practice, diminishing returns would require a nonlinear model.

Properties of the OLS Estimator

The Gauss-Markov Theorem

Under assumptions 1-5 (linearity, independence, homoscedasticity, no multicollinearity, exogeneity -- normality is NOT required), the OLS estimator is BLUE: Best Linear Unbiased Estimator.

1

Linear: \hat{\boldsymbol{\beta}} is a linear function of \mathbf{y}: \hat{\boldsymbol{\beta}} = \mathbf{C}\mathbf{y} where \mathbf{C} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T.

2

Unbiased: E[\hat{\boldsymbol{\beta}}] = E[(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}] = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{X}\boldsymbol{\beta} = \boldsymbol{\beta}.

3

Best (Minimum Variance): \text{Var}(\hat{\boldsymbol{\beta}}) = \sigma^2(\mathbf{X}^T\mathbf{X})^{-1}. For any other linear unbiased estimator \tilde{\boldsymbol{\beta}}, the matrix \text{Var}(\tilde{\boldsymbol{\beta}}) - \text{Var}(\hat{\boldsymbol{\beta}}) is positive semi-definite.

Climate Trend Application

Fitting a linear trend to global temperature anomalies (1880-present): \text{Anomaly} = \beta_0 + \beta_1 \cdot \text{Year}. The OLS estimate \hat{\beta}_1 \approx 0.008 degrees C per year is the BLUE -- no other linear unbiased procedure produces a tighter confidence interval for the warming trend.

PySpark and ETL Fundamentals

A hands-on course covering PySpark and ETL fundamentals. Each module builds on the previous one using a shared student enrollment dataset.

  • Module 1: ETL with PySpark
    Extract, transform, and load data using PySpark pipelines.
  • Module 2: In-Memory Data Processing
    Understand why Spark keeps data in memory and how it differs from disk-based processing.
  • Module 3: Spark RDDs
    Work with Resilient Distributed Datasets -- Spark's foundational abstraction.
  • Module 4: DataFrames: PySpark vs Pandas
    Compare the same operations in both APIs side-by-side.
  • Module 5: Querying with PySpark SQL
    Register DataFrames as tables and query them with SQL syntax.
  • Module 6: UDFs in PySpark
    Write and apply User-Defined Functions for custom transformations.
  • Module 7: Data Cleaning
    Handle nulls, duplicates, inconsistent formats, and invalid values.
  • Module 8: Data Exploration
    Summarize, profile, and understand your cleaned dataset.
  • Module 9: Transformation Operations
    Reshape, join, aggregate, and pivot data for analysis.
  • Module 10: Caching and Broadcast Variables
    Optimize your Spark pipeline with caching strategies and broadcast joins.

Hypothesis Testing

Interactive modules covering the full hypothesis testing workflow -- from defining H0 and H1 through paired t-tests. View full track page

  • Module 1: Foundations of Hypothesis Testing
    The decision framework, null and alternative hypotheses, test statistics, one- and two-tailed tests.
  • Module 2: Errors, Alpha, p-value, and Critical Value Method
    Type I/II errors, significance level, power, p-value interpretation, critical value approach.
  • Module 3: One-Sample Z-Test
    Testing a population mean with known sigma. Full derivation, step-by-step procedure, confidence intervals.
  • Module 4: Two-Sample Tests and A/B Testing
    Two-population Z-test, Welch's t-test, A/B testing, Cohen's d effect size.
  • Module 5: Paired t-Test
    Dependent samples, before-after designs, reduction to one-sample t on differences.

K-Means Clustering

From the intuition of grouping similar things to the mathematics of centroids and silhouette scores. View full track page

  • Module 1: Why Clustering? Intuition and the K-Means Idea
    Unsupervised learning setting, partitioning, real-world applications, animated K-means demo.
  • Module 2: Distance, Means, and Centroids
    Euclidean, Manhattan, cosine distance. The centroid as the minimiser of squared distances. Mathematical connection to the mean.
  • Module 3: Inertia (WCSS) and the Elbow Method
    WCSS as the K-means objective, coordinate descent view, elbow method, and its limitations.
  • Module 4: Silhouette Score and Evaluation Metrics
    Silhouette formula and interpretation, silhouette plots, Davies-Bouldin, Calinski-Harabasz, practical guide to choosing K.

Decision Trees and Ensemble Methods

From splitting a single node to training gradient-boosted forests. View full track page

  • Module 1: Decision Trees — Intuition and Structure
    Why trees, how prediction works, tree anatomy, and the decision boundary as axis-aligned cuts.
  • Module 2: Impurity Measures and the Training Process
    Gini impurity, entropy, misclassification rate, information gain, and the CART algorithm.
  • Module 3: Gini Index Splits
    Full worked example, threshold search, continuous vs categorical features, Gini vs entropy.
  • Module 4: Advantages, Limitations, and Pruning
    Interpretability strengths, variance and instability failures, pre-pruning and cost-complexity pruning.
  • Module 5: Bagging and Random Forests
    Bootstrap aggregating, variance reduction derivation, random feature subsets, feature importance, OOB error.
  • Module 6: Gradient Boosting
    Functional gradient descent, pseudo-residuals, shrinkage, XGBoost second-order approximation.
  • Module 7: Hyperparameter Tuning
    Every knob for RF and GB explained through the bias-variance lens with interactive simulators.

Natural Language Processing

From raw text to language models — 10 modules covering lexical, syntactic, and semantic NLP with interactive spaCy and NLTK widgets. View full track page

  • Module 1: Text Normalization & Edit Distance
    Tokenization, lowercasing, stopword removal, and Levenshtein distance with interactive DP matrix widget.
  • Module 2: Stemming, Lemmatization & Noisy Channel
    Porter/Snowball stemmers, lemmatization, and the Bayesian noisy channel model for spell correction.
  • Module 3: POS Tagging & Shallow Parsing
    Penn Treebank tags, HMM-based tagging, NP chunking, and live color-coded POS tagger widget.
  • Module 4: CFG, Parse Trees & Deep Parsing
    Context-free grammars, constituency trees, structural ambiguity, SVG step-through parser.
  • Module 5: Lexical Semantics & WSD
    WordNet synsets, hypernyms, polysemy vs homonymy, Lesk algorithm step-through.
  • Module 6: Distributional Semantics & PMI
    Co-occurrence matrices, PMI/PPMI formulas, and live Plotly heatmap from editable corpus.
  • Module 7: NER, IOB, CRF, SRL & Coreference
    Named entities, BIO tagging, CRF sequence models, semantic roles, and coreference chains.
  • Module 8: BoW, TF-IDF & Topic Modeling
    Bag of words, TF-IDF formula, IDF weighting, NMF topic modeling with Plotly heatmap.
  • Module 9: Word Embeddings
    word2vec skip-gram/CBOW, embedding space, cosine similarity, analogy tasks, BERT overview.
  • Module 10: N-gram Language Models & Applications
    Chain rule, Laplace smoothing, perplexity, text generation, summarization, and MT overview.

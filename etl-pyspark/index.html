PySpark and ETL Fundamentals

A hands-on course covering PySpark and ETL fundamentals. Each module builds on the previous one using a shared student enrollment dataset.

  • Module 1: ETL with PySpark
    Extract, transform, and load data using PySpark pipelines.
  • Module 2: In-Memory Data Processing
    Understand why Spark keeps data in memory and how it differs from disk-based processing.
  • Module 3: Spark RDDs
    Work with Resilient Distributed Datasets -- Spark's foundational abstraction.
  • Module 4: DataFrames: PySpark vs Pandas
    Compare the same operations in both APIs side-by-side.
  • Module 5: Querying with PySpark SQL
    Register DataFrames as tables and query them with SQL syntax.
  • Module 6: UDFs in PySpark
    Write and apply User-Defined Functions for custom transformations.
  • Module 7: Data Cleaning
    Handle nulls, duplicates, inconsistent formats, and invalid values.
  • Module 8: Data Exploration
    Summarize, profile, and understand your cleaned dataset.
  • Module 9: Transformation Operations
    Reshape, join, aggregate, and pivot data for analysis.
  • Module 10: Caching and Broadcast Variables
    Optimize your Spark pipeline with caching strategies and broadcast joins.
